# Dashboard Help Center Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a drawer-based help center to the dashboard with a topbar entry, five help tabs, and content that reuses live strategy/config metadata while keeping the operator on the current page.

**Architecture:** Extend the existing single-file dashboard HTML/CSS/JS generator in `dashboard.py` rather than introducing a new route or template system. Reuse the existing `/api/config` metadata for strategy and field explanations, and render the help drawer entirely on the client side with small, focused render helpers.

**Tech Stack:** Python 3.10, stdlib HTTP server, inline HTML/CSS/JavaScript generated by `dashboard.py`, pytest

- [ ] **Step 4: Update `renderHelpDrawer()` to render the selected tab**

Implement:

```javascript
function renderHelpDrawer() {
  const backdrop = el('helpBackdrop');
  const drawer = el('helpDrawer');
  const tabs = el('helpTabs');
  const body = el('helpBody');
  const footer = el('helpFooter');

  backdrop.classList.toggle('open', state.helpOpen);
  drawer.classList.toggle('open', state.helpOpen);
  drawer.setAttribute('aria-hidden', state.helpOpen ? 'false' : 'true');

  tabs.innerHTML = HELP_TABS.map((tab) => {
    const active = tab.id === state.helpTab ? ' help-tab-active' : '';
    return '<button class="help-tab' + active + '" data-help-tab="' + esc(tab.id) + '" type="button">' + esc(tab.label) + '</button>';
  }).join('');

  body.innerHTML = state.helpTab === 'quickstart'
    ? renderHelpQuickStart()
    : renderHelpPageGuide();

  footer.innerHTML = '';
  tabs.querySelectorAll('[data-help-tab]').forEach((node) => {
    node.addEventListener('click', () => {
      state.helpTab = node.getAttribute('data-help-tab') || 'quickstart';
      renderHelpDrawer();
    });
  });
}
```

```javascript
function renderHelpConfigDictionary() {
  const payload = state.config || {};
  const groups = payload.field_groups || [];
  const help = payload.field_help || {};
  const scope = payload.field_scope || {};

  return groups.map((group) => {
    const items = (group.keys || []).map((key) => {
      const scopeNote = scope[key] === 'strategy_5_only' ? '仅策略 5 重点使用' : '所有策略可见';
      return '<li><strong>' + esc(formatConfigLabel(key, payload.labels || {})) + '</strong>' +
        '<div class="help-item-subkey">' + esc(key) + '</div>' +
        '<div>' + esc(help[key] || '暂无说明') + '</div>' +
        '<div class="help-item-scope">' + esc(scopeNote) + '</div></li>';
    }).join('');
    return '<section class="help-section"><h3>' + esc(group.title || '参数分组') + '</h3><ul class="help-detail-list">' + items + '</ul></section>';
  }).join('');
}
```

```javascript
function renderHelpStrategyGuide() {
```
